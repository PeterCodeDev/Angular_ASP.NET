**MovieVerse** (Angular Películas) es una plataforma web completa diseñada para los amantes del séptimo arte. Permite explorar los últimos estrenos, consultar detalles técnicos de películas, conocer la trayectoria de actores y gestionar un catálogo completo de cine.

---

## 🍿 Funcionalidades Destacadas

* **Explorador de Películas:** Listado dinámico con filtros por género, fecha de estreno y popularidad.
* **Fichas de Actores:** Perfiles detallados de los artistas, incluyendo biografía y filmografía completa.
* **Gestión de Catálogo (CRUD):** Interfaz administrativa para añadir, editar o eliminar películas y actores de la base de datos.
* **Buscador Predictivo:** Sistema de búsqueda en tiempo real para localizar títulos o intérpretes al instante.
* **Galería Multimedia:** Visualización de pósters, backdrops y trailers en alta calidad.

---

## 🚀 Características Técnicas

* **Consumo de API REST:** Integración con servicios externos para obtener datos actualizados del mundo del cine.
* **Reactive Forms:** Validaciones avanzadas para el registro de nuevas películas y actores.
* **Lazy Loading:** Optimización de carga por módulos para una navegación ultra rápida.
* **Servicios Centralizados:** Gestión eficiente del estado de la aplicación y la comunicación entre componentes.
* **Pipes Personalizados:** Formateo de datos específicos como duraciones de películas, presupuestos y fechas.
